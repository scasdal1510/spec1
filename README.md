# Ejemplo de context engineering utilizando SPEC

## Contenido del proyecto
### agents.md
**Configura los roles,las capacidades y los permisos de las IAs que trabajan en el repositorio**
### constitution.md
**Define los principios innegociables, el tono de la comunicacion y las directrices éticas o de seguridad.**
### architecture.md
**Describe la estructura del sistema, el flujo de datos, la integracion de servicios y el stack tecnologico**
### decision.md
**Registrar el porqué de las decisiones tomadas en el pasado para evitar cilcos infinitos de cambios**
### spec.md
**Define qué problemas resuelve la aplicacion y qué funcionalidades debe tener. Es  el puente entre el negocio y el código**
